# Dependencies
**N.B** clone zsh plugins into ~/.zsh
- `git`
- `zsh`
    - `zsh-autosuggestions`
    - `zsh-history-substring-search`
    - `zsh-syntax-highlighting`
- `wget`


# Quick Install
```
cd ~ && wget https://raw.githubusercontent.com/ayumodark/zshrc/refs/heads/config/.zshrc && exec zsh
```
